# _Wallet Project_

This wallet program is implemented in four different languages: 

> Javascript, Java, Python and Go

###  How to execute it in __JAVASCRIPT__  :

The project structure is organized as follows:

```tree

└───main.js (Entrypoint)
└───walletfeatures.js (Implementation of the features)
└───auth.js (Authentication implementation)
 
```

* You just have to execute the main.js to launch the program.

To access the wallet, enter these identifiers:

```tree
Nom d'utilisateur: random
Mot de passe: strongpassword
```
